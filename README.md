## Installation:
Install nest
```bash
$ npm i -g @nestjs/cli
```
Create NestJs Project with name's project
```bash
$ nest new project-name
```
Create node modules
```bash
$ npm install
```
Install TypeOrm e MySQL
```bash
$ npm install --save @nestjs/typeorm typeorm mysql2
```
